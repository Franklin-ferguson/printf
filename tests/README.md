This directory contains tests for src files at the root directory.

Determining which test file refers to what src file.
Format:
src file	- 0-abcde.c
test file	- zero.c

Example
src file		test file
1-printf.c		one.c

For Several, test files, maintain "one" + "number++"
Look at the example below:

src file		test file
2-score.c		two-1.c
			two-2.c
			two-3.c
3-app.c			three-1.c
			three-2.c
